
Sintaxe opcional e simplificada para percorrer coleções

sintaxe:
``` java
String[] vect = new String[] {"Maria", "Bob", "Alex"};  
  
//for loop  
for(int i = 0; i < vect.length; i++) {  
    System.out.println(vect[i]);  
}  
  
// for-each loop  
for(String nome : vect) {  
    System.out.println(nome);  
}
```
