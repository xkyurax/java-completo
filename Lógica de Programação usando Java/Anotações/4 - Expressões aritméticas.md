
**Expressões aritméticas**
É toda expressão cujo cálculo resulta em um valor numérico

**Operadores aritméticos**
```
+: adição
-: subtração
*: multiplicação
/: divisão
%: resto da divisão("mod")

Precedência:
1º lugar: * / %
2º lugar: + -

(3 + 2) * 4
a precedência do que está em parênteses é maior
```
